# keel
[构建中,请勿使用]  keel 是基于react.js的脚手架，可以一键生成react项目的环境。

# construct

```
.
├── dist
│   ├── app
│   └── static
└── src
    ├── app // 项目程序入口
    ├── page  // 项目页面入口
    └── static  // 项目静态资源入口
```
